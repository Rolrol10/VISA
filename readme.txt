Dette repositoriet skal som formål gjøre det mulig å kontrollere labutstyr som strømforsyninger og elektronisk last via kode.

Python scriptet setup.py skal installere pakken man trenger og åpne nettsiden hvor man finner driveren for VISA protokollen.

Dersom scriptet ikke virker:

Man trenger VISA protokoll driver, som man finner her:
https://www.ni.com/en/support/downloads/drivers/download.ni-visa.html#558610

Man trenger pyvisa pakken:
pip install pyvisa
